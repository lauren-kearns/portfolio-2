E-LEARNING

Restoring the Connection

Overview

Restoring the Connection is a scenario-based eLearning module designed for call centre agents in the telecoms industry, focused on handling customer connectivity issues effectively.

My Role

eLearning Developer

Instructional Designer

Tools

Articulate Storyline

Canva

ChatGPT

Problem / Goal

Customer Support Agents need to resolve technical issues while managing customer frustration, but training is often too theoretical and not applied to real conversations. The goal was to create a realistic, scenario-based experience that improves decision-making, communication, and problem resolution.

Experience It
Process
01
Analyse Identified performance gaps, learner needs, and key challenges in handling customer connectivity issues.
02
Design Defined learning objectives, structured the scenario flow, and mapped decision points to reflect real call interactions.
03
Develop Created the storyboard, wrote content, and designed visuals to support an engaging, scenario-based experience.
04

05

Outcome

An interactive, scenario-based learning experience that simulates real customer calls, helping agents build confidence and apply skills in a practical, low-risk environment. The design supports behaviour change by improving how agents handle customer interactions—encouraging clearer communication, more effective problem-solving, and reduced reliance on escalations.
